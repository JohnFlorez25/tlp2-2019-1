#Listando Componentes

##Importante uso de Map
Para evidenciar como sería el proceso de mostrar una lista de perfiles, hacemos uso de la opción para a manipulación de arrays, map, con el fin de capturar un objeto de un conjunto de datos previos (datos quemados: datos mook), esto nos ayuda a mostrar un entorno de simulación siendo el caso que recibieramos nuestros datos desde una API externa, así ya tendríamos conocimiento en el lugar que recibimos los datos de nuestro backend
